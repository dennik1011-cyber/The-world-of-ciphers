REVERSE CIPHER GAME

Описание:
Это простая консольная игра на Python.
Игроку показывается текст, написанный наоборот.
Нужно расшифровать сообщение.

Пример:

ПРИВЕТ -> ТЕВИРП

Как запустить:

1. Открыть проект в PyCharm
2. Запустить файл main.py
