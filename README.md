# 🔐 The World Of Ciphers

**Разгадай. Взломай. Победи.**

Веб-приложение с 20 уникальными шифрами — от простого переворота строки до шифра Вернама. Тренируй криптографическое мышление, зарабатывай рейтинг и соревнуйся с другими игроками.

---

## 🎮 Геймплей

1. **Выбери уровень** — 20 шифров разбиты по сложности: easy, normal, medium, hard
2. **Увидь шифротекст** — каждое слово генерируется случайно, так что уровни можно переигрывать
3. **Введи ответ** — расшифруй и получи очки
4. **Прокачай рейтинг** — за каждый правильный ответ начисляются баллы:
   - Easy: 50
   - Normal: 100
   - Medium: 150
   - Hard: 200

| Технология | Применение |
|-----------|-----------|
| **Flask** | Веб-фреймворк |

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/dennik1011-cyber/The-world-of-ciphers.git
cd The-world-of-ciphers

# Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

# Установить зависимости
pip install -r requirements.txt

# Запустить
python app.py
```

Открыть в браузере: `http://127.0.0.1:5000`
